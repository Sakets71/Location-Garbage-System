# Location Garbage System

##  Overview

The **Location Garbage System** is a web-based application designed to streamline waste management processes. Built using HTML, CSS, JavaScript, and PHP, this system allows users to report garbage-related issues, enabling authorities to address them efficiently.

##  Features

- **User Registration & Login**: Secure access for users to report issues.
- **Issue Reporting**: Users can submit details about garbage-related problems.
- **Admin Dashboard**: Administrators can view and manage reported issues.
- **Driver Interface**: Drivers can access assigned tasks and update statuses.

##  Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL

## Getting Started

### Prerequisites

- PHP 7.4 or higher
- MySQL
- A local server environment like XAMPP or WAMP

### Usage

**Users:** Register and log in to report garbage-related issues.  
**Administrators:** Manage user reports and assign tasks to drivers.  
**Drivers:** View assigned tasks and update their status upon completion.
